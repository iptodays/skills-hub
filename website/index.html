核心功能

一切，都在本地运行

01
浏览与搜索
卡片列表展示所有安装在 ~/.agents/skills 下的技能，支持按分类筛选、关键词实时搜索名称 / 描述 / 标签。
02
Markdown 文档渲染
内置渲染器，直接阅读每个技能的 SKILL.md，无需离开应用，代码块高亮、表格、blockquote 全部支持。
03
AI 深度分析
调用 Claude API（可自定义 Base URL 与模型）对技能进行深度分析。结果永久缓存在本地 SQLite，内容不变则不消耗 API 额度。
04
Git 克隆安装
从 GitHub / GitLab / Gitee 一键克隆安装，支持带路径的子目录安装。安装前展示影响说明，需二次确认。
05
安全删除
删除前展示详细影响列表（目录永久删除、Agent 立即不可用、引用 prompt 可能失效），需二次确认方可执行。
06
系统托盘驻留
关闭窗口后驻留系统托盘，单击托盘图标唤起主窗口，随时可用，不占用 Dock 空间。
